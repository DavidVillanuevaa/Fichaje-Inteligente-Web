# Fichaje Semanal

## Uso

1. Abre `index.html` en el navegador.
2. Elige idioma en el selector superior (`Español` o `English`).
3. Elige el año del calendario activo.
4. Selecciona una tarea existente o crea una nueva desde `Tarea`.
5. Crea fichajes indicando fecha, hora de inicio y fin.
6. Navega entre semanas con los botones de semana anterior/siguiente.

## Características

- Selector de idioma bilingüe (ES/EN).
- Selector de calendario activo para cualquier año.
- Control por semanas ISO (lunes a domingo) del año seleccionado.
- Resumen semanal por tarea (horas y minutos).
- Total anual por tarea (horas dedicadas en el año activo).
- Gestión de tareas: añadir, editar y borrar desde un listado.
- Gráfico comparativo de tareas (horas por tarea del año activo).
- Cronómetro de fichaje: iniciar, pausar/reanudar, finalizar y guardar automáticamente.
- Edición de fichajes guardados haciendo clic en la semana y ajustando horas de inicio/fin.
- Datos guardados en una base de datos local del navegador (`IndexedDB`).
